## JSON to shell variable converter

### Usage

```
$ echo '{"foo": [1, 2, 3, 8.923878], "bar": {"baz": {"a": "b"}}}' | js2sh
FOO_0="1"
FOO_1="2"
FOO_2="3"
FOO_3="8.923878"
BAR_BAZ_A="b"
```
